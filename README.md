## How to run

1. Clone the repository in your `catkin_ws/src` directory.
2. Run `source ~/catkin_ws/devel/setup.*sh` command dependent on your shell.
3. Compile the code:
    ```shell
    cd ~/catkin_ws
    catkin_make
    ```
